## Summary

- Added configurable auto-refresh interval (default 5 minutes) for prayer time updates.
- After the last prayer of the day, the countdown now targets the next day's Fajr.
- Improved reliability with HTTPS API calls and clearer error logging.
- Status coloring begins 20 minutes before prayer and minute-by-minute alerts appear in the final 10 minutes.

## Installation

### Using VS Code UI

- Download the attached `prayer-reminder-1.0.6.vsix` file from the release assets.
- Open the Command Palette (`Ctrl` + `Shift` + `P` on Windows/Linux, `Cmd` + `Shift` + `P` on macOS).
- Run **Extensions: Install from VSIX...** and select the downloaded package.

### Using the CLI

```bash
code --install-extension prayer-reminder-1.0.6.vsix
```
